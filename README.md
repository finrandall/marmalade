Marmalade is a type of fruit preserve similar to jam, but usually made from citrus fruit - often including the peel - whereas jam is made from peeled and crushed non-citrus fruits.

Marmalade is a framework for atomistic spin dynamics (ASD) simulation for those left with a bitter taste after C++. This repository contains a developing Python workflow for simulating spin systems using the Landau-Lifshitz-Gilbert equation.
